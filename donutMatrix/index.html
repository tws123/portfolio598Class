Assignment 4 | Creativity Exploration 1

Description

This exploration uses the p5.js library to created and rotate 3-dimensional shapes (spheres and tori) using WEBGL. Web Graphics Library is a JavaScript API for rendering 3D graphics. At this time, p5.js 2D elements (like text) can't be used in a WEBGL 3D space; there are, therefore, no instructional buttons in this program. Click the mouse to see differnt shape configurations.
